# LDDTools.lol Hub + Bulk POD Resizer

This is a small Vite + React + TypeScript + Tailwind project that powers:

- `/` – a white-card landing page for LDDTools.lol
- `/resize` – a Bulk POD Resizer page that uses your POD Resize-tab presets

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`

Deploy-ready for Netlify and GitHub Pages (SPA redirect is configured in `netlify.toml`).
